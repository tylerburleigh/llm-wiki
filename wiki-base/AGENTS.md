# Codex Instructions

This vault's canonical operating schema lives in `CLAUDE.md`. Read and
follow `CLAUDE.md` first; it defines the wiki layout, claim typing,
frontmatter contracts, ingest/query/repair/lint workflows, and the
deterministic `wiki-ops` command layer.

The `.agents/` directory contains Codex-facing workflow briefs. They are
generic markdown instructions. Use them as entry points for ingest,
query, lint, repair, purpose review, extraction, and audit work.
